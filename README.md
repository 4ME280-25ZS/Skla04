# Jednoduchá kontaktní stránka

Tento repozitář obsahuje jednoduchou statickou stránku s kontaktními informacemi.

Soubory:
- `index.html` — hlavní stránka (nahraďte zástupné texty skutečnými kontakty)
- `styles.css` — základní styly

Jak spustit lokálně:

```bash
# v kořenovém adresáři projektu
python3 -m http.server 8000
# pak otevřete http://localhost:8000 v prohlížeči
```

Návod k úpravě kontaktů:
- Otevřete `index.html` a nahraďte `ZDE_VLOZIT_JMENO`, `ZDE_VLOZIT_TELEFON`, `ZDE_VLOZIT_EMAIL`, `ZDE_VLOZIT_ADRESU` skutečnými hodnotami.

Pokud chcete, mohu tyto hodnoty doplnit za vás — pošlete mi prosím kontaktní údaje.
